# 概要
C++ライフゲーム

# 必要なもの
* libncursesw
* libncurses
* CMake
* g++

# 使い方
ビルドしてできた実行ファイルを実行するだけ
終了は `q`

# オプション
```
-h, --help                                ヘルプを表示
-v, --version                             バージョンを表示
--height=height                           高さ height に設定 
--width=width                             幅 width に設定
-p=probability, --probability=probability 生きているセルの生存確率を設定 (0 <= probability <= 1)
```
